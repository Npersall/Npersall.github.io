#[Npersall.github.io](Npersall.github.io)

This website is a project that was assigned as a project during Phase 0 of
Dev Bootcamp. 

The assignment was to choose something you love such as a hobby or experience and build a small website around it. 

The criteria was to have  a 2 page site that gave information about your chosen hobby and to have fun with it and explore HTML and CSS.

I choose to do mountain biking and and introduction to the world of bikes.

The site was originally built with basic HTML and CSS. I have gone back and updated it using Sinatra and RESTful conventions.
